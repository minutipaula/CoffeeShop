# Specialty Coffee E-Commerce Storefront

A modern high-performance and fully responsive landing page for a specialty coffee shop. Designed with a strong focus on typography, color contrast compliance and smooth user interactions.

- Features
**Interactive Cart System:** Real-time item count updates and visual badge feedback.
**Dynamic Navigation:** Seamless product category switching (Coffee vs. Brewing Machines).
**Responsive Layout:** Optimized for mobile, tablet, and desktop viewports.
 **Modern Typography:** Styled using `Syne` for headings and `Plus Jakarta Sans` for clean, readable body copy.

- Tech Stack
 **HTML5** - Semantic structure
 **CSS3** - Custom layouts, animations, and accessible color palettes
**JavaScript (Vanilla)** - Cart state management and DOM manipulation
